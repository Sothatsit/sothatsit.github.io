Hi, I'm Padraig Lamont

I've always enjoyed programming, and so I tend to do it a lot.

I'm currently a student at UWA. A particular interest of mine is creating and releasing diverse projects. I am also very interested in the nature of human knowledge and memory. Some projects I'm proud of are listed below!

Notable Projects

  • I created RoyalUr.net, a website where people can play the Royal Game of Ur. The Royal Game of Ur is an ancient Mesopotamian racing board game, that dates back to 2500BC. It is considered to be a predecessor to Backgammon. As of writing, there are about 400 games played every day on RoyalUr.net. This project is open-source.
  • I was a part of creating, and I continue to maintain, The Misinformation Game, a social media simulator built to study people's behaviour when they interact with social media. This project is also open-source.
  • I am currently working on Tyche. Tyche is a Python library that allows the modelling of beliefs as knowledge bases of entities, with probabilistic beliefs about the entities, and probabilistic relationships between them. It contains functionality to query these belief models using aleatoric description logic. It also contains a novel approach to learning the probabilities within belief models based upon observations, which has showed promise. It's a bit technical, but I really like it.
  • I am also currently working with a team on PubMed Connections. PubMed Connections is a website to allow researchers to explore the relationships between articles, authors, and journals in PubMed, through visualising them in a network.
